# **Devil Connection Mod Loader (汎用MODローダー)**

[English](http://docs.google.com/README.md)  |  [繁體中文](https://www.google.com/search?q=README.zh-TW.md)

これは、ゲーム\*\*『でびるコネクショん』(Devil Connection)\*\* 専用に設計された非公式の汎用MODローダーです。

本ツールを使用することで、**オリジナルの巨大なファイルを解凍・置換することなく**、外部マウント方式で .asar 形式の言語パック、修正パッチ、機能モジュールを読み込むことができます。このアーキテクチャにより、パッチサイズを大幅に削減し、**ゲーム作者およびMOD制作者の著作権をより適切に保護**します。

## **✨ 機能と特徴**

* **非破壊インストール**: HDiffPatch技術を使用してコア部分のみに最小限の注入を行い、オリジナルファイルのバックアップを保持するため、いつでも復元可能です。  
* **プラグインシステム**: plugins フォルダ内の .asar ファイルまたはフォルダを自動的に読み込みます。  
* **優先順位管理**: 複数のMODの同時読み込みをサポートし、ファイル名による上書き順序の制御が可能です。  
* **超軽量**: コアパッチのサイズは極めて小さく（数KB程度）、ゲームのパフォーマンスに影響を与えません。

## **🚀 インストール方法**

以下の手順に従ってインストールしてください。

1. **本プロジェクトのダウンロード**:  
   * GitHubの \[Releases\] から最新バージョンの圧縮ファイルをダウンロードしてください。  
2. **パッチツールのダウンロード**:  
   * [HDiffPatch Releases](https://github.com/sisong/HDiffPatch/releases) にアクセスし、最新版をダウンロードしてください。  
   * 解凍して hpatchz.exe を見つけてください（hdiffz ではなく **hpatchz** ですのでご注意ください）。  
3. **ファイルの配置**:  
   * 本プロジェクトのファイル (Loader.js, patch.hdiff, install\_mod\_loader.bat) と、ダウンロードした hpatchz.exe を、すべてゲームのルートディレクトリにコピーしてください。

**📂 ファイル構成例:**SteamLibrary\\steamapps\\common\\でびるコネクショん\\  
│  
├── Loader.js                \<-- ローダー本体  
├── patch.hdiff              \<-- コア差分パッチ  
├── install\_mod\_loader.bat   \<-- インストールスクリプト  
├── hpatchz.exe              \<-- パッチ適用ツール  
└── DevilConnection.exe      \<-- (オリジナルのゲーム実行ファイル)

4. **インストールの実行**:  
   * **install\_mod\_loader.bat** をダブルクリックして実行します。  
   * スクリプトが自動的にオリジナルファイルのバックアップを作成し、ローダーを注入します。  
   * 成功すると、ゲームディレクトリ内に plugins フォルダが生成されます。  
5. **完了**: これでMODのインストール準備が整いました。

## **📖 使用方法**

### **🟢 基本的な使用法：MOD/翻訳のインストール**

1. 使用したいMODファイル（通常は .asar 形式、例：zh-tw.asar）を入手します。  
2. そのファイルをゲームルートディレクトリ下の **plugins** フォルダに入れます。  
3. ゲームを直接起動すると、MODが自動的に適用されます。

### **🟠 高度な使用法：マルチMODの優先順位**

本ローダーは複数のMODの同時読み込みをサポートしており、ファイル名の降順 (Z \-\> A) で優先順位が決定されます。  
数字が大きい、またはアルファベットが後ろのファイルほど、優先順位が高くなります（前のファイルを上書きします）。  
例:  
plugins フォルダに以下のファイルがあると仮定します：

* 01\_Base\_Translation.asar (基本翻訳)  
* 99\_UI\_Fix.asar (UI修正)

読み込み結果:  
ローダーは先に 99\_UI\_Fix.asar を読み込み、その後に 01\_Base\_Translation.asar を読み込みます。  
両方のMODが同じ画像を変更している場合、ゲームには 99\_UI\_Fix.asar のバージョンが表示されます。  
これにより、翻訳ファイル全体を再パッケージ化することなく、「修正パッチ」や「互換パック」を簡単に作成できます。

## **⚠️ 注意事項と削除方法**

* **ゲームのアップデート**: Steamがゲームをアップデートした場合、ローダーのコアが無効になる可能性があります。その場合は、再度 install\_mod\_loader.bat を実行して修復してください（スクリプトがバックアップと更新を自動的に処理します）。  
* **MODの削除**: plugins フォルダから対象の .asar ファイルを削除または移動するだけです。  
* **ローダーの削除**:  
  1. Loader.js と plugins フォルダを削除します。  
  2. Steamの「ファイルの整合性を確認」機能を使用して、app.asar を復元します。  
  3. または、手動で app.asar を削除し、バックアップファイルの app.asar.bak を app.asar にリネームします。
